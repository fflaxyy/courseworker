 Курсовая работа: поиск и первичный анализ наборов данных

Репозиторий содержит Python-код для выполнения практической части курсовой работы по теме:

«Поиск и первичный анализ наборов данных по различным областям применения»

В работе рассматриваются четыре типа данных:

1. табличные данные;
2. временные ряды;
3. изображения;
4. текстовые данные.

В репозитории размещены только файлы с кодом. Сами датасеты не загружаются в GitHub, так как они берутся из открытых источников Kaggle и могут иметь большой объём.



 Структура репозитория

text
courseworker/
│
├── code_chapter_1.py
├── code_chapter_2.py
├── code_chapter_3.py
├── code_chapter_4.py
└── README.md


После запуска кода автоматически создаются папки с результатами и графиками.



 Используемые датасеты

 1. Табличные данные

Датасет: Diamonds Prices
Источник: Kaggle / классический набор данных diamonds
Задача: прогнозирование цены бриллианта по его характеристикам
Файл для запуска: `code_chapter_1.py`

Ссылка на датасет:
https://www.kaggle.com/datasets/nancyalaswad90/diamonds-prices

Для первой главы можно использовать файл `diamonds.csv`. Если файла рядом со скриптом нет, код использует встроенный датасет `diamonds` из библиотеки `plotnine`.



 2. Временные ряды

Датасет: NVIDIA Daily Stock Prices Dataset
Источник: Kaggle
Задача: прогнозирование цены закрытия акций NVIDIA
Файл для запуска: `code_chapter_2.py`

Ссылка на датасет:
https://www.kaggle.com/datasets/ibrahimshahrukh/nvidia-daily-stock-prices-20162026-dataset/data

Для запуска нужно скачать CSV-файл и положить его рядом со скриптом.
Ожидаемое имя файла:

text
NVDA_yfinance_clean.csv


Если файл называется иначе, можно переименовать его или изменить переменную `DATA_FILE` в начале файла `code_chapter_2.py`.



 3. Изображения

Датасет: Animal faces dataset (AFHQv2 512x512)
Источник: Kaggle / StarGAN v2
Задача: многоклассовая классификация изображений животных
Классы: `cat`, `dog`, `wildlife`
Файл для запуска: `code_chapter_3.py`

Ссылка на датасет:
https://www.kaggle.com/datasets/dimensi0n/afhq-512

Для третьей главы используется облегчённая версия кода, потому что полный датасет AFHQv2 содержит около 15 000 изображений и занимает много места. Загружать полный набор изображений в GitHub не требуется.

Код не обрабатывает весь датасет целиком. Он строит основные графики и таблицы по описанию набора данных, а для визуальных примеров использует небольшую выборку изображений.

При первом запуске скрипт создаёт папки:

text
sample_afhq_examples/
├── cat/
├── dog/
└── wildlife/


Чтобы на графике появились реальные примеры изображений, нужно скачать датасет с Kaggle и вручную положить по нескольку изображений в соответствующие папки:

text
sample_afhq_examples/cat/
sample_afhq_examples/dog/
sample_afhq_examples/wildlife/


После этого нужно повторно запустить файл:

bash
python code_chapter_3.py


Результаты сохраняются в папку:

text
figures_chapter3_light/


Если изображения не добавлены, код всё равно запускается и создаёт графики с поясняющими заглушками.



 4. Текстовые данные

Датасет: Trip Advisor Hotel Reviews
Источник: Kaggle
Задача: анализ тональности отзывов об отелях
Файл для запуска: `code_chapter_4.py`

Ссылка на датасет:
https://www.kaggle.com/datasets/andrewmvd/trip-advisor-hotel-reviews

Для запуска нужно скачать CSV-файл и положить его рядом со скриптом.
Ожидаемое имя файла:

text
tripadvisor_hotel_reviews.csv


Если файл называется иначе, можно переименовать его или изменить переменную `DATA_FILE` в начале файла `code_chapter_4.py`.



 Установка библиотек

Перед запуском рекомендуется установить Python версии 3.10 или выше.

Необходимые библиотеки:

bash
pip install pandas numpy matplotlib seaborn scikit-learn statsmodels scipy pillow plotnine wordcloud


Если используется виртуальное окружение, его можно создать так:

bash
python -m venv venv


А затем активировать:

Для Windows:

bash
venv\Scripts\activate


Для Linux / macOS:

bash
source venv/bin/activate


После активации нужно установить библиотеки:

bash
pip install pandas numpy matplotlib seaborn scikit-learn statsmodels scipy pillow plotnine wordcloud




 Как запускать код

Каждый файл запускается отдельно.

 Глава 1 — табличные данные

bash
python code_chapter_1.py


После запуска создаётся папка:

text
chapter1_figures/


В ней сохраняются графики для анализа табличного датасета Diamonds Prices.



 Глава 2 — временные ряды

Перед запуском рядом со скриптом должен находиться файл:

text
NVDA_yfinance_clean.csv


Запуск:

bash
python code_chapter_2.py


После запуска создаётся папка:

text
figures_chapter2/


В ней сохраняются графики по временным рядам акций NVIDIA.



 Глава 3 — изображения

Для облегчённого запуска полный датасет скачивать необязательно.

Запуск:

bash
python code_chapter_3.py


После первого запуска создаются папки для примеров изображений:

text
sample_afhq_examples/cat/
sample_afhq_examples/dog/
sample_afhq_examples/wildlife/


Чтобы получить график с реальными примерами классов, нужно положить в эти папки несколько изображений из датасета AFHQv2 и повторить запуск:

bash
python code_chapter_3.py


Результаты сохраняются в папку:

text
figures_chapter3_light/


Облегчённая версия используется потому, что полный датасет изображений имеет большой объём, а для курсовой работы достаточно показать структуру классов, баланс, формат изображений и выборочные примеры.



 Глава 4 — текстовые данные

Перед запуском рядом со скриптом должен находиться файл:

text
tripadvisor_hotel_reviews.csv


Запуск:

bash
python code_chapter_4.py


После запуска создаётся папка:

text
figures_chapter4_final_5figures/


В ней сохраняются основные графики для анализа текстовых данных.



 Результаты работы кода

Код выполняет первичный анализ наборов данных:

* проверку структуры данных;
* анализ пропусков и дубликатов;
* построение графиков распределений;
* анализ выбросов;
* корреляционный анализ;
* анализ временного ряда;
* анализ баланса классов изображений;
* предобработку текста;
* частотный анализ;
* TF-IDF;
* поиск похожих текстов.

Полученные графики используются в курсовой работе и презентации.



 Примечание по датасетам

Датасеты не добавлены в репозиторий из-за их размера и условий использования. Для воспроизведения результатов необходимо скачать данные с Kaggle по ссылкам выше и положить нужные CSV-файлы рядом с соответствующими Python-скриптами.

Для третьей главы полный датасет изображений загружать в репозиторий не требуется. Достаточно использовать небольшую выборку изображений для демонстрации примеров классов.
